PWN

PWN is the unironic name of a project I want to undertake and add to during my time here in CSH. PWN is similar to Metasploit in idea, but focuses on alerting us upon exploitation rather than just exploitation. Sound interesting? Check out Metasploit here!

Current modules supported right now are single and ranged SSH server password bruteforcer. Here, you can specify IPs on the CSH network that you want to test the credibility of SSH passwords. For more information, please see the readme page here.
